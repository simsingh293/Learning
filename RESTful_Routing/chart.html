RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /lions GET List all lions Lion.find()
New /lions/new GET Show new lion form N/A
Create /lions POST Create new lion, then redirect somewhere(index) Lion.create()
Show /lions/:id GET Show info about one specific lion Lion.findById()
Edit /lions/:id/edit GET Show edit form for one lion Lion.findById
Update /lions/:id PUT Update particular lion, then redirect somewhere Lion.findByIdAndUpdate()
Destroy /lions/:id DELETE Delete a particular lion, then redirect somehwere Lion.findByIdAndRemove()
